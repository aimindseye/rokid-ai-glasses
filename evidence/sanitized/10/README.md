# Test 10 Sanitized Evidence

This directory contains reviewed public derivatives only.

Included:

- service-family summaries,
- local-package file metadata and hashes,
- offline execution conclusions, and
- Local-versus-Online architecture comparison.

Excluded:

- packet captures,
- TLS keylogs,
- decrypted payloads,
- Android and application logs,
- screenshots,
- model binaries,
- account and device identifiers,
- Bluetooth addresses,
- exact locations, and
- private filesystem paths.

Private archive provenance is recorded separately in
[`../../manifests/10-private-evidence-sha256.json`](../../manifests/10-private-evidence-sha256.json).
