# Community Ecosystem

The Rokid ecosystem includes display-equipped AR glasses, the display-free
Style model, older products, phone companions, server bridges, AIUI agents, and
experimental tools.

## Broad discovery index

[awesome-rokid](https://github.com/Anezium/awesome-rokid) is the recommended
broad index across products and project categories.

This repository has a narrower role: a consumer, developer, and independently
validated guide specifically for the non-display Style model.

## Why compatibility is difficult

A project may require:

- a Micro-LED HUD;
- touchpad/D-pad navigation;
- YodaOS APK installation;
- CXR rendering APIs;
- China-only services;
- a specific firmware build;
- partner/developer access.

A project can be useful as a reference without being usable on Style.

## Most relevant Style categories

- phone companion apps;
- audio and notification bridges;
- camera/media transfer;
- voice agents;
- Bluetooth controllers;
- privacy/network tooling;
- firmware/device-management research.

## Adding a resource

Record:

1. target product;
2. display requirement;
3. phone OS;
4. SDK/API;
5. installation method;
6. region/account dependency;
7. Style validation status.
